Publication

    M. Kurvey, M. Pawaskar, S. Nikam, R. Jagtap, S. Bhandary and A. Acharya, “Establishing Mesh Network to Transfer and Visualize Data for Safety of Underground Miners”, published in IEEE Xplore Digital Library, Oct’23, pp. 1134-1139, doi: 10.1109/ICPCSN58827.2023.00192.
    IEEE Published paper Link
    A safety monitoring and smart system are essential in the mining industry to avoid any kind of adverse occurrence and to enhance miners' safety and efficiency. This might be accomplished by creating a wearable that comprises several sensors that detect safety parameters and transmit the data to the network. The research's primary goal is to create a wireless system for coal miners utilizing a LAN network employing numerous ESP modules in a mesh topology. This system will allow centralized management to monitor real-time environmental parameters and take preemptive steps. This may be done using third-party software or an offline website. The proposed work provides a real-time ESP-based wireless multi-node network monitoring system that is technologically advanced, with minimal latency, high accuracy, and high reliability when compared to the available wired monitoring system for underground mines. In addition to that, the work also provides a simplified interface for the display of acquired data.
